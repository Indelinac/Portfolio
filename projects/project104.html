#Contexte

Le projet est lié au cours de base de données. Nous étions 4 pour réaliser ce projet sur une durée d’un mois environ.

#Objectif

Nous devions livrer une base de données sur une entreprise fictive ; de la conception à la mise en œuvre avec un code SQL. Ainsi, parallèlement, nous devions étudier les données de celle-ci pour nous sensibiliser sur le cycle de la donnée et ses impacts.

#Réalisation

Nous avons rendu une base de données satisfaisante sur une entreprise de concessionnaire automobile nommé Wheel’s R Us. Ce fut compliqué au début car je ne connaissais pas ce milieu mais avec l’aide de mes camarades et de nombreuses recherches, nous avons réussi à mettre en place un MCD et un site web basée sur cette entreprise fictive.
